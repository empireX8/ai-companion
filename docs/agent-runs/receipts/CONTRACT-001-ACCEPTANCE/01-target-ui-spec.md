# Target UI Spec - CONTRACT-001 Acceptance

**Date:** 2026-06-29

## Surface

This slice does not change UI layout. The target is the serialized What Changed / reality-tracking report output produced by the shared builder.

## Required Visible Behavior

- Identity-label language is rejected as a conclusion.
- The report uses the active reality-gate phrasing when evidence is insufficient.
- The report requests timestamped behavioral receipts rather than identity confirmation.
- Fresh output does not emit new `mixed` evidence labels.

## Explicit Non-Goals

- No screenshot-driven redesign
- No inspector styling work
- No route-dependent workaround
- No UI-only shim for bad output
