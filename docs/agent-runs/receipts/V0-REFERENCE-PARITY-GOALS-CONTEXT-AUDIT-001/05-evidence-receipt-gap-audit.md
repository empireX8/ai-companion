# Evidence Receipt Gap Audit

## Stronger Surfaces
- What Changed surfaces movement plus evidence, disconfirming evidence, and re-entry conditions.
- Today surfaces state, delta log, evidence pointers, and re-entry triggers.
- Active Questions pages expose competing theories, evidence needed, and connected map items.

## Weak Spots
- The Context page shows memory statements and pattern counts, but not the receipts or disconfirming evidence behind the derived context layer.
- Generic goal capture stores a statement and type, but no evidence linkage or change condition is surfaced next to the goal itself.
- The Map context rail presents labels and counts, but production selection does not drill into evidence for those items.
- Watch For uses an unavailable footer label for Active Questions instead of pointing the user to the real question flow.

## Conclusion
- Evidence-path coverage is good for movement surfaces and weak for derived context and goal surfaces.
