# Visual Scope

The goal of this pass was to compare the current production Orvek workbench against the v0/reference visual direction:

- light-first private intelligence workbench
- restrained and evidence-led
- no therapy, journal, productivity, or chatbot wrapper feel
- no generic SaaS softness
- no purple-heavy direction
- no graph-first gimmickry
- no over-friendly empty states

The reference shell and page screenshots were used as the visual baseline.
